### 单独使用方法

1、下载本项目

2、将下载后的压缩包中**CN文件夹**，放入游戏目录的：**游戏目录/media/lua/shared/Translate**中（如果不放心，可以提前备份游戏目录中的**CN文件夹**）

3、运行游戏
